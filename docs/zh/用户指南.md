# 1 注册中心用户指南

## 1.1 特性介绍

注册中心是一个专注于Agent统一管理的服务，支持用户将来自不同厂商的Agent进行集中注册与管理，实现多源Agent的可控接入与维护。

### 1.1.1 应用场景

应用于A2A-T领域多厂商多智能体交互场景下的AgentCard统一注册和管理。

### 1.1.2 能力范围

1. 本项目提供Agent注册中心模块供客户系统集成，用于管理客户系统内部的Agent，提供Agent注册、Agent查询能力。

2. 默认所注册的Agent会作为公共资源，暂无Agent所有者设计。

3. 本项目仅用作功能模块，非完整系统，模块自身不提供登录认证、鉴权、用户管理、日志审计、加解密、秘钥管理、数据库等能力，需由客户系统提供如上安全基础设施；源码中已预留相关方法函数，供二次定制实现。

4. 本项目对注册的AgentCard信息默认使用文件存储：data/agentcard.json，测试场景下可以手动修改文件，重启服务生效。

### 1.1.3 设计约束
1. 本项目需要运行在linux系统上，支持ipv4环境

2. 当前支持单实例部署，仅用于内部系统，不可开放到公网，不可作为云服务部署，否则目标系统需要同步提供防火墙、提供web服务器实现认证鉴权等安全能力。

3. 向本项目注册的AgentCard不可含有个人数据例如电话号码，不可含有敏感信息例如密码、凭据，否则有信息泄露风险。

## 1.2 能力详细介绍

### 1.2.1 管理能力（CLI）

#### 1.2.1.1 启动CLI

#### 1.2.1.2 Agent管理

管理员：agent查询（查全量、查详情）、agent审核、agent分类

#### 1.2.1.3 标签管理

管理员：对标签的增删改查

#### 1.2.1.4 证书&公钥管理

管理员：注册中心验签公钥/证书的配置、签名公钥的展示与下载
管理员：注册中心配置对端TLS证书的信任列表

### 1.2.2 接口能力

- **注册AgentCard**：支持将不同厂商的Agent注册到中心，统一纳管。
- **查询AgentCard列表**：根据指定条件查询符合条件的AgentCard列表。
- **查询指定AgentCard**：按AgentCard名称和组织精确查找唯一的AgentCard实例。
- **更新指定AgentCard**：更新指定AgentCard的信息。
- **删除指定AgentCard**：删除不再使用的AgentCard。
- **按语义检索AgentCard**：根据自然语言语义检索相匹配的AgentCard。

### 1.2.3 安全能力

#### 1.2.3.1 HTTPS通信

通信的身份安全、协议安全、算法安全、校验对端证书等满足安全要求

#### 1.2.3.2 校验Agent签名

默认校验签名，可配关闭；
支持导入验签公钥/证书；
支持从签名中获取验签公钥/证书；
验证agentCard签名字段

#### 1.2.3.3 提供注册中心签名

使用Agent注册中心自己的私钥为AgentCard签名

#### 1.2.3.4 Prompt注入检测

1. 注册阶段：
   - 校验各入参字段sche- ma   
   - 利用开源组件LLM Guard或Rebuff做黑名单规则校验
2. 检索阶段（看有没有小模型可用检索）：
   - 输入：结构隔离 
   - 输出：结果白名单约束

#### 1.2.3.5 高危Tool检查

- 根据Agent携带的Tool特征，定义风险等级以及检查策略，例如：
- 严重：delete_resource、access_credentials、shell_exec
- 高：write_file、send_email
- 中：list_directory
- 低：search_wiki
- 并制定不同的策略

#### 1.2.3.6 人工审核

支持配置开启人工审核：AgentCard注册时完成基础校验后，对于高风险AgentCard信息进行人工确认后再发布

提供AgentCard的合规、功能、法律等要求的人工审核机制，审核通过后发布，包括不限于以下维度：
1. 法律与数据合规维度
2. 业务逻辑与质量维度
3. 资源与成本维度
4. 注入等攻击的人为确认

#### 1.2.3.7 操作隔离：Agent owner

A注册的Agent，B不能修改、删除；
AgentCard按agent所有者隔离，仅支持所有者修改、删除自己的AgentCard

#### 1.2.3.8 检索范围控制：Agent分类

给Agent标记分类标签，检索结果按分类隔离，控制检索范围（在于效率提升而不是检索范围？）

#### 1.2.3.9 日志审计

提供重要操作的记录审计日志
