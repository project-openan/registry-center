# 通信证书生成系统设计文档

## 1. 需求概述

本系统提供一个外部接口，用于生成通信身份证书。该证书用于保障通信过程中的身份认证和数据安全。

### 1.1 功能需求
- 提供外部接口调用生成通信身份证书
- 支持证书的自动创建和管理
- 支持证书目录的自动创建和校验

### 1.2 安全需求
- **身份安全**：确保证书与使用者的身份绑定，防止身份冒用
- **协议安全**：使用安全的通信协议，防止中间人攻击
- **算法安全**：使用业界认可的安全加密算法
- **校验对端证书**：支持对对端证书的有效性进行校验

## 2. 架构设计

### 2.1 系统架构
```
外部接口层
    ↓
证书生成服务层
    ↓
证书生成工具层
    ↓
文件系统 + 审计日志
```

### 2.2 核心组件
- **外部接口**：提供 HTTP/HTTPS 接口供外部调用
- **CertificateGenerator**：证书生成工具类
- **证书存储**：文件系统存储证书文件
- **审计日志**：记录证书生成操作

## 4. 接口设计

### 4.1 证书生成接口

**请求**
请求方信息，用于生成证书
```
POST /rest/a2a-t/v1/certificates/generate
Content-Type: application/json

{
  "certificate_id": "string",  // 证书标识
  "validity_period": 365,      // 有效期（天）
  "key_size": 2048,            // 密钥长度
  "algorithm": "RSA"          // 加密算法
}
```

**响应**
```
{
  "code": 200,
  "message": "success",
  "data": {
    "certificate_id": "string",
    "certificate_path": "/path/to/certificate.crt", // 证书
    "signing_key_path": "/path/to/signing.key", //签名密钥
    "encryption_key_path": "/path/to/encryption.key", // 加密密钥文件
    "password_path": "/path/to/password.txt", // 加密口令文件
    "generated_at": "2026-04-08T10:00:00Z",
    "expires_at": "2027-04-08T10:00:00Z"
  }
}
```

## 5. 实现细节

### 5.1 证书生成流程

```
1. 开始
   ↓
2. 校验证书目录是否存在
   ↓
3. 目录不存在 → 创建证书目录
   ↓
4. 判断证书是否存在
   ↓
5. 证书不存在 → 开始创建证书
   ↓
6. 生成签名密钥
   ↓
7. 生成服务器证书文件
   ↓
8. 生成加密密钥文件
   ↓
9. 生成加密口令文件
   ↓
10. 记录审计日志
    ↓
11. 返回证书文件信息
    ↓
12. 结束
```

### 5.2 文件结构

```
certificates/
├── {certificate_id}/
│   ├── certificate.crt          # 服务器证书
│   ├── signing.key              # 签名密钥
│   ├── encryption.key           # 加密密钥
│   └── password.txt             # 加密口令
└── audit.log                    # 审计日志
```

## 6. 安全考虑

### 6.1 身份安全
- 使用证书绑定使用者身份信息
- 证书包含唯一标识符
- 支持证书吊销机制

### 6.2 协议安全
- 强制使用 HTTPS 协议
- 支持 TLS 1.2 及以上版本
- 禁用不安全的加密套件

### 6.3 算法安全
- 签名算法：RSA-2048 或 ECDSA-P256
- 加密算法：AES-256-GCM
- 哈希算法：SHA-256
- 密钥交换：ECDHE

### 6.4 证书校验
- 校验证书有效期
- 校验证书链完整性
- 校验证书吊销状态
- 校验证书主题名称

### 6.5 密钥安全
- 私钥文件权限设置为 600
- 密钥使用强密码保护
- 支持密钥轮换机制
- 密钥存储使用加密保护

## 7. 异常处理

### 7.1 异常类型
- `CertificateDirectoryError`: 证书目录操作失败
- `CertificateGenerationError`: 证书生成失败
- `KeyGenerationError`: 密钥生成失败
- `AuditLogError`: 审计日志记录失败

### 7.2 错误码
- `CERT_DIR_CREATE_FAILED`: 证书目录创建失败
- `CERT_ALREADY_EXISTS`: 证书已存在
- `KEY_GEN_FAILED`: 密钥生成失败
- `CERT_GEN_FAILED`: 证书生成失败
- `AUDIT_LOG_FAILED`: 审计日志记录失败
